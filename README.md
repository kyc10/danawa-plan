# 💰 가계부 대시보드

엑셀 가계부 파일을 업로드하면 항목별 월 소비 추이를 대시보드 형태로 시각화하는 웹 애플리케이션입니다.

## 주요 기능

- 📊 **전체 월별 지출 추이** - 꺾은선 차트 + 평균선
- 🍩 **항목별 비중** - 도넛 차트
- 📦 **항목별 누적 막대** - 카테고리별 월 소비 분포
- 📈 **카테고리별 월 추이** - 탭 선택으로 항목별 상세 보기
- 🗺️ **지출 히트맵** - 카테고리 × 월별 강도 시각화
- 💡 **인사이트 & 절감 제안** - 자동 분석 (대출이자, 교육비, 외식비 등)
- 🔍 **거래 내역 검색** - 월/카테고리/텍스트 필터
- 🌙 **라이트/다크 모드** 토글
- 📂 **엑셀 파일 업로드** - 새 월 데이터 자동 반영

## 실행 방법

```bash
# 의존성 설치
pip install flask flask-cors openpyxl pandas

# 서버 실행
python server.py
```

브라우저에서 `http://localhost:5000` 접속

## 엑셀 파일 형식

시트명: `YYMM` (예: `2501` = 2025년 1월)

각 시트 구조:
- 3행: 헤더 (이용일자, 이용내역, 이용금액, 분류)
- 4행~: 카테고리 합계 (B열: 카테고리명, C열: 금액) + 거래 내역 (F~J열)

지원 카테고리: 식료품, 꾸밈비, 외식비, 문화비, 의료비, 공과금, 교육비, 교통비, 보험료, 차량비용, 여행경비, 대출이자, 경조사비, 생필품, 기타

## 기술 스택

- **Backend**: Python Flask
- **Frontend**: Vanilla JS, Chart.js 4
- **Data**: openpyxl, JSON
